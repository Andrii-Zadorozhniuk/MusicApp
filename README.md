This is a personal learning project — a music streaming app built to practice and improve my Android development skills.

The app includes features such as music playback, artist and album browsing, search, favorites, and history. It is built using modern technologies like Kotlin, Jetpack Compose, MVVM architecture, Room, Paging 3, and ExoPlayer/Media3.

The main goal of this project is to apply real-world development patterns, experiment with clean architecture, and gain hands-on experience with building scalable and maintainable applications.
